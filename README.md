# MindGarden - 基于自然疗愈的学生压力干预AI平台

## 项目简介

MindGarden是一个基于自然疗愈理论的学生压力干预AI平台，旨在帮助学生缓解压力、改善心理健康。平台结合了现代Web技术、AI技术和自然疗愈理论，为学生提供科学、便捷的压力管理工具。

## 功能特色

### 🧠 AI压力评估
- 科学的压力评估问卷
- DeepSeek AI大模型智能分析
- 个性化的压力状况报告
- 实时AI对话功能

### 🌿 自然干预实验室
- 森林浴疗法
- 正念冥想
- 水疗法
- 芳香疗法
- 音乐疗法

### 🌍 全球压力地图
- 全球学生压力状况展示
- 数据可视化和趋势分析

### 🎨 自然主题设计
- 绿色、蓝色等自然色彩
- 响应式设计，支持移动端
- 自然元素装饰

## 技术架构

### 前端技术
- **HTML5 + CSS3 + JavaScript** - 原生Web技术
- **响应式设计** - 支持各种设备
- **现代CSS** - Flexbox、Grid布局
- **Font Awesome** - 图标库

### AI集成
- **DeepSeek API** - 压力评估分析和智能对话
- **结构化提示词** - 优化AI响应质量

## 快速开始

### 1. 直接使用
1. 下载 `index.html` 文件
2. 在浏览器中打开 `index.html`
3. 开始使用平台功能

### 2. 配置API密钥
1. 编辑 `config.js` 文件
2. 将 `DEEPSEEK_API_KEY` 替换为您的实际API密钥
3. 在 `index.html` 中引入 `config.js`

### 3. 部署到GitHub Pages
1. 将文件上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为源
4. 访问 `https://yourusername.github.io/repository-name`

## 使用说明

### 压力评估
1. 点击"压力评估"进入评估页面
2. 填写基本信息（姓名、年龄、性别、职业）
3. 完成5道压力评估问题
4. 输入压力情况描述（至少50字）
5. 提交评估，获得AI分析结果
6. 查看个性化干预建议
7. 使用智能对话功能进一步咨询

### 干预实验室
1. 点击"干预实验室"查看各种自然疗法
2. 了解每种疗法的科学依据
3. 选择适合的干预方案
4. 按照建议进行实践

### 全球地图
1. 点击"全球地图"查看全球压力数据
2. 了解不同地区的压力状况
3. 查看干预效果统计

## API配置

### DeepSeek API设置
1. 访问 [DeepSeek官网](https://www.deepseek.com/) 注册账号
2. 获取API密钥
3. 在 `config.js` 中配置API密钥

### API使用限制
- 请遵守DeepSeek API的使用条款
- 注意API调用频率限制
- 保护API密钥安全
