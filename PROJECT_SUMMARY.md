## 🚀 使用方法

### 方法1：直接打开
1. 双击 `index.html` 文件
2. 在浏览器中打开即可使用

### 方法2：本地服务器（推荐）
1. 使用Live Server等工具运行
2. 访问 `http://localhost:5500`

### 方法3：GitHub Pages部署
1. 上传文件到GitHub仓库
2. 启用GitHub Pages
3. 访问 `https://yourusername.github.io/repository-name`

## ⚙️ API配置

### 配置DeepSeek API
1. 编辑 `config.js` 文件
2. 将 `DEEPSEEK_API_KEY` 替换为您的实际API密钥
3. 保存文件

### API功能
- **压力评估分析**：基于用户问卷数据生成个性化分析
